HTA Help

The first step of HTA is to orient all of the edge cubies so they are solvable without using quarter-twists of the front or back layers, see a description of the goal state here. Press the Random button and view the status of each cubie in the table above the cube (Y=correct, N=incorrect, FL=Front Left position, etc). The status will update after each move. If the whole cube is rotated while solving then press the Repos button to return the cube to the original position so cubie locations match the status.

The only way to change N cubies to Y is with a quarter-twist of the front or back layers which toggles the state of all four cubies. Move N cubies to the front or back layer and fix four at a time. If two N cubies remain then move them to the locations shown in one of the rows in the table below and use the sequence.
