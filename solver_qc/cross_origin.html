Shared Memory for Web Workers
Cross-Origin Headers

To enable shared memory for web workers set the Cross-Origin headers in your local web server as shown in the example below which is for the Apache web server on Fedora. Consult your platform specific documentation for the details of where to set these headers on your system.

File: /etc/httpd/conf.d/userdir.conf

<Directory "/home/*/public_html">
...
Header set Cross-Origin-Opener-Policy same-origin
Header set Cross-Origin-Embedder-Policy require-corp
</Directory>

Restart the web server after making changes.
