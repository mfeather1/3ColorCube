Quad-Core Solver
Run Locally

To run the Quad-Core Solver locally, download the 3ColorCube-master.zip file from Gitub at the link below and extract the files into your local web directory.

https://github.com/mfeather1/3ColorCube/archive/refs/heads/master.zip

Add the following headers to the configuration for your web server, if using Apache an alternative is to create file .htaccess in the 3ColorCube-master directory with these lines:

Header set Cross-Origin-Opener-Policy same-origin
Header set Cross-Origin-Embedder-Policy require-corp

Restart your web server and navigate to your local copy of the 3ColorCube web site via localhost or 127.0.0.1 to run the Quad-Core Solver.

If the following message appears in cube area:
"Your organization has blocked access to this page or website"
then press ctrl-F5 (or ctrl-shift-r) to reload the page.
