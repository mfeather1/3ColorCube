Advanced Solving

This is for the case that has six misplaced facelets where it is only possible to fix one as shown on the first cube below (case means this configuration and any others that can be made with half-twists). For the complete solution Sequence 3C-E2 is used four times with the first two fixing one facelet each. The second cube shows how the Alternate Setup for Sequence 3C-E1 can be used to solve with one less sequence. This case has one cubie with both facelets misplaced (one "double" and four "singles") but the distinguishing factor is that only one facelet can be fixed because there is a similar case where it is possible to fix two facelets (in which case that should be done instead).
