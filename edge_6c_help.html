This demo shows how to solve the edges on a 6-color cube by first using the 3-Color Edge Sequences and then using the 6-Color Edge Sequences. The solution table shows the number of moves in each step which alternate between making setups (with half-twists or rotation of the cube) and applying sequences. It is not always possible to match a setup exactly, in those cases the number of facelets that match the setup is shown next to the sequence name.
