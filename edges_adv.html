3-Color Edges
Advanced Solving

The following shows how Sequence 3C-E3 can be modified to handle two pairs of cubies (second pair is in bottom layer) by adding a half-twist of the front face on the fifth move. This sequence can be used whenever there are two pairs of cubies (each with both facelets misplaced) in two different slices, make the setup with half-twists and/or cube rotation where applicable. The color of the individual facelets does not have to match the setup, they just have to be misplaced. At least four facelets will be fixed, likely more (see table below).



With the above setup, the rest of the cube does not have to be already solved so any four of the twelve cubies can be included. Hence there are 81 possible configurations of the four cubies where each has both facelets misplaced. The following table shows the result of using the sequence on all of them.
