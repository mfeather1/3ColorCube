3x3 Cube Symmetry Counts

In the table, the Mins column is the number of unique configs when reduced by symmetry (using rotation and reflection, a "min" is the config with the lowest numerical value of all that are equivalent by these operations hence each min represents up to 48 different configs). Clicking on any of the links will generate counts for the designated cube component. In the result table there is a "Syms" column which is the number of configs that can be generated by applying rotation and reflection to a Min, thus Mins * Syms = Configs on each line. This differs from the number of symmetries which is actually 48/Syms.
