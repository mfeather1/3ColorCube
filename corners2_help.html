The cubes in this set are the 17% that are HTO solvable. For each cube, an optimal solution is shown which is always four moves or less (and there may be more than one way to do it). However, it may be easier to learn how to solve using a method that may take more moves than an optimal solution such as with this solution method.

For the other 83% of cubes, a sequence can be used to get an HTO configuration as shown here. The full set of cubes is available in Interactive Cubes as "6-Color Corners: Pre-Solved as 3-Color (ALL)".
