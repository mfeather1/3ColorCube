Solver 2
All Phase 2 Sequences

This slightly modified version of the solver loads all sequences for phase 2 which allows instant solving for any of the four million presolved cubes.

For random cubes, it shows more solutions in the log which are the ones excluded by the eight move limit for phase two in the regular version of the solver. However, those solutions are at the higher end of the range in terms of overall length which means that they are of no benefit if searching for solutions with 20 moves or less. If searching for instant solutions (with search time set to 0) then it may in some cases provided a faster response albeit with more moves.
