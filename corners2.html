Waterwheel Configuration

This configuration has the interesting feature that twisting the left or right face has no effect. Use the mouse to turn the left or right face on either cube below and see for yourself. This config is easily indentified by the horizontal color pairs which alternate all the way around the cube (blue, yellow, blue, yellow, ...). You can see how it resembles a Waterwheel on the Starter Cube as shown here.

These two cubes are actually the same as can be seen by rotating either of them horizontally. Press Play on each cube to see how to solve. After the first move, the blue facelets on both of these cubes match the setup for the second cube on the Starter page. This configuration is used to solve the corners on the 6-color cube.
